# Papernuko-Lang
HSPでプログラミング言語っぽいものを作る**テスト**。  
## 現在実装されている命令・関数
* print命令
<pre><font color="skyblue">print</font> &lt;text&gt;</pre>
* locate命令
<pre><font color="skyblue">locate</font> &lt;x&gt;,&lt;y&gt;</pre>
* title命令
<pre><font color="skyblue">title</font> &lt;text&gt;</pre>
## 特徴
* なんと文字列は"で囲わなくてもいい！
